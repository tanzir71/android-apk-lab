# SMS Wake-on-LAN APK landing page

Static GitHub Pages subpage for SMS Wake-on-LAN inside android-apk-landing-pages.

## Files

- index.html - SEO landing page and APK install guide
- assets/sms-wake-lan.apk - release APK
- assets/sms-wake-lan-screenshot.png - app screenshot used in the hero
- styles.css, site.js - shared technical-sheet styling inspired by tinyproctor.js
- robots.txt, sitemap.xml, llms.txt - crawler and AI-agent context

## Deploy

This folder is copied into the master android-apk-landing-pages repo. Enable GitHub Pages from the master repo root.

Expected URL: https://tanzir71.github.io/android-apk-landing-pages/sms-wake-lan/
