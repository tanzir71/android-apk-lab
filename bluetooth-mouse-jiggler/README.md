# Bluetooth Mouse Jiggler APK landing page

Static GitHub Pages subpage for Bluetooth Mouse Jiggler inside android-apk-landing-pages.

## Files

- index.html - SEO landing page and APK install guide
- assets/bluetooth-mouse-jiggler.apk - release APK
- assets/bluetooth-mouse-jiggler-screenshot.jpg - app screenshot used in the hero
- styles.css, site.js - shared technical-sheet styling inspired by tinyproctor.js
- robots.txt, sitemap.xml, llms.txt - crawler and AI-agent context

## Deploy

This folder is copied into the master android-apk-landing-pages repo. Enable GitHub Pages from the master repo root.
