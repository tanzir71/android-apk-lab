# Tanzir Android APK Lab

Single GitHub Pages repo containing all APK landing pages and downloadable APK artifacts.

## Structure

- index.html - master app directory
- styles.css, site.js - shared static styling and copy helpers
- bengal-ledger/ - Bengal Ledger page, APK, screenshot, checksum
- bluetooth-mouse-jiggler/ - Bluetooth Mouse Jiggler page, APK, screenshot, checksum
- memoryos/ - MemoryOS page, APK, screenshot, checksum
- pocket-crm/ - Pocket CRM page, APK, screenshot, checksum
- promiseqr/ - PromiseQR page, APK, screenshot, checksum
- sms-geolocate/ - SMS Geolocate page, APK, screenshot, checksum
- sms-wake-lan/ - SMS Wake-on-LAN page, APK, screenshot, checksum

## Deploy

Push this repo to GitHub, then enable GitHub Pages from main / root.

Expected URL: https://tanzir71.github.io/android-apk-lab/
