LAB / 02

Sideloading requires explicit permission

Android usually blocks unknown-source APKs until you enable Developer options and allow the browser, Files app, or download manager to install apps from outside Google Play.

Developer mode

  1. Open Settings, then About phone. On some phones, open Software information.
  2. Tap Build number seven times and enter your lock screen PIN if Android asks.
  3. Return to Settings and open Developer options. Leave unrelated settings alone.

Unknown sources

  1. Download an APK from one of the app pages.
  2. Open Settings, Apps, Special app access, Install unknown apps.
  3. Allow the browser or Files app you used, install the APK, then turn that allowance off again.
